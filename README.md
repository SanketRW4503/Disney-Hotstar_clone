# Disney+Hotstar Powered by Gemini

## Overview

**Disney+Hotstar Powered by Gemini** is a frontend web application that replicates the Disney+Hotstar streaming platform, enhanced with AI-driven search functionality powered by Gemini. Users can explore a wide range of movies and shows and enjoy a seamless experience.

## Features

- **AI-Powered Search**: Quickly find movies using the Gemini AI search functionality.
- **User-Friendly Interface**: Similar UI to Disney+Hotstar.

## Technologies Used

- **Frontend**: 
  - React.js
  - Tailwind CSS
  - Redux 
  - React Router DOM
  - Webpack

- **Backend Services**:
  - Firebase for authentication
  - Gemini for efficient search
